완성 범위
-상품목록 출력
-장바구니 널값 입력 및 출력
-상품을 장바구니에 넣기
-장바구니 품목 통합 계산
-쇼핑몰 종료

에러 부분
- 장바구니 넣을 품명 입력시 입력한 값이 널값으로 입력된다.

문제해결
: 원도우 10 설정중 UTF-8 체크,
 원도우 터미널을 다운로드 받아 설치하고 vs코드로 작성된 파일을 실행 시키자 문자열과 정수가 널값으로 반환 되는 오류를 해결 하였습니다.
